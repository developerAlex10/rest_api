Требуется автоматизировать тестирование rest api сервиса:

1. Зарегистрировать нового юзера c минимально допустимыми параметрами;
2. Добавить этому пользователю игру;
3. Добавить пользователю DLC к игре из п2;
4. Создать еще 1 дополнительный тест по сценарию на выбор
После прохождения теста удалить в конце созданного юзера из БД

Требования к проекту:
- Опубликовать решение в отдельном репозитории на GitHub;
- Использовать java 17/21, gradle, RestAssured, Allure
- Использовать DTO(Pojo) классы для маппинга Json
- Тесты должны быть перезапускаемыми

Будет плюсом:
- Создать еще 2 простых теста на выбор;
- Запустить тесты многопоточно;
- Использовать Junit Extensions
